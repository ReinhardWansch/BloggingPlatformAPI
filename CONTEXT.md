
## aktuelles Issue: Projektgrundlage mit Node.js und Express aufsetzen
### Ziel
Die Basis des Backends soll sauber eingerichtet werden.

### Aufgaben
- Node.js-Projekt initialisieren
- Express installieren
- Startskript anlegen
- Grundstruktur für src, routes, controllers und config vorbereiten

### Akzeptanzkriterien
- Das Projekt startet lokal ohne Fehler
- Ein einfacher Server läuft auf einem konfigurierbaren Port
- Die Ordnerstruktur ist übersichtlich und verständlich

---

## Hirnpresse

Man könnte bei einem Push nach main automatisch von einem Agenten ein Refactoring nach den Clean Code Regeln durchführen lassen. Anschließend soll automatisch ein Pull-Request mit den vorgeschlagenen Änderungen inklusive Erklärung erstellt werden.
